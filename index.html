MailsterSMTP

A SMTP server API written in Java

Download this project as a .zip file Download this project as a tar.gz file

MailsterSMTP is a SMTP server API written in Java. It is a NIO fork of the SubEthaSMTP project since the premature end of it's 2.x branch. It is notably used in the Mailster project core.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout edeoliveira/mailstersmtp
